---

## Система "АгроКонтроль"

Разработана автоматизированная система управления производством для предприятия по производству средств защиты растений (гербициды, инсектициды, фунгициды).

### Модули системы:

1. **AgroControlAPI** — бэкенд-сервер, обеспечивающий хранение данных, бизнес-логику и API
2. **TechnologistModule** — рабочее место технолога (управление рецептурами и техкартами)
3. **LaboratoryModule** — рабочее место лаборанта (контроль качества сырья и продукции)
4. **OperatorModule** — рабочее место аппаратчика (выполнение технологических операций)
